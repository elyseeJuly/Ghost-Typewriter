## 四大对抗模块回顾

### 1. ⚔️ 对抗生成集群 (Adversarial Cluster)
- **功能**: 并行调用 5 套底层风格指令（语音速记、失眠碎碎念、微博口语等）。
- **入口**: 输入框上方的 ⚔️ GAN 按钮。
- **效果**: 从 5 个变体中筛选 AI 特征最弱、最拟人的版本。

### 2. 🔬 判别器沙盒 (Discriminator Sandbox)
- **功能**: 对生成变体进行实时打分。支持 GPTZero API (需配置 Key) 或本地启发式算法。
- **意义**: 让生成器在“实战环境中”进化，选出真正的优胜者。

### 3. 〰 向量示波器 (Vector Scope)
- **功能**: 浏览器端 Sentence-BERT 语义分析。
- **反馈**: 以 🟢/🟡/🔴 热力图标注语义死区。
- **建议**: 指引用户在死区插入“感官记忆碎片”以撕裂 AI 模型生成的平滑向量。

### 4. ⏱ 时间戳伪造引擎 (Temporal Forge)
- **功能**: 自动反推长达数小时的非线性编辑时间轴。
- **导出**: 支持导出含 `<!-- Edit v1.x -->` 的 Markdown，确保证据链完美。
