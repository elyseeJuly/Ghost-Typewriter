# Ghost Typewriter v9.0 — Adversarial GAN Edition & GitHub Sync

将现有的"Hybrid Detox Edition"升级为具备 GAN 对抗思维的完整作战平台。核心要素：生成对抗集群（5路并行生成 + 判别器打分）、向量示波器（客户端 Sentence-BERT 热力图）、时间戳伪造引擎（伪造编辑历史）。

---

## ⚠️ User Review Required

> [!IMPORTANT]
> **向量示波器（Vector Scope）技术方案裁决**
> `@xenova/transformers` + `all-MiniLM-L6-v2` 需要在浏览器端 WebAssembly 加载约 **25~80MB** 的模型文件，首次启动耗时较长（取决于网络/缓存）。有两个方案：
> - **方案 A（PRD 指定）**：完全客户端推理，首次加载慢，但完全私密、零服务器成本。
> - **方案 B（降级）**：用语义启发式算法替代（基于词汇重叠+句长方差模拟相似度），无模型加载开销，即时响应，但精度较低。
>
> **默认采用方案 A（PRD 原文要求），但首次向量分析前会显示模型下载进度 UI。**

> [!IMPORTANT]
> **判别器反馈回路（Discriminator Loop）API 接入**
> PRD 提及接入 GPTZero / Originality.ai API。这些 API 均需付费密钥。计划为：
> - 在 API Vault 中增加 **GPTZero API Key** 字段（可选），有则真实调用，无则使用**本地模拟判别器**（基于现有 X-Ray 扫描分数反向换算）作为 fallback，让用户也能体验流程。
> - 导出 Prompt 优胜记录到本地。

> [!WARNING]
> **UI 布局重大变化**：v9.0 引入了四阶段流水线，现有三栏布局需要重构。计划采用**带标签页切换的扩展式面板**，在不破坏现有三列骨架的前提下，在中列引入「对抗生成集群」模式，右侧引入「判别器结果面板」可折叠区域。大版本视觉标题也会从"Hybrid Detox Edition"升级为"Adversarial GAN Edition"。

### 2. 全新核心库

#### [NEW] `src/lib/vectorScope.ts`
负责向量示波器（Vector Scope）完整逻辑：
- 懒加载 `@xenova/transformers` pipeline（`feature-extraction`, `Xenova/all-MiniLM-L6-v2`）
- 导出 `analyzeSemanticVariance(text, onProgress?)` → 返回每个句子的余弦相似度数组
- 导出 `buildHeatmapData(similarities)` → 返回带颜色权重的热力图数据（0=绿, 0.85+=红）
- 导出 `MODEL_LOAD_STATES` 枚举（idle / loading / ready / error）

#### [NEW] `src/lib/discriminator.ts`
判别器沙盒预检逻辑：
- 导出 `runDiscriminatorCheck(text, apiKey, gptzeroKey?)` → `{ score: number, verdict: string }`
  - 有 GPTZero key: 真实 API 调用
  - 无 key: 本地 fallback（X-Ray 分数 + 句子平均长度启发式合成 AI 率 0~100）
- 导出 `PromptStrategyPool`（存储优胜 Prompt）及 `recordWinner(input, promptStyle, score)` 
- LocalStorage 持久化优胜样本库

#### [NEW] `src/lib/temporalForge.ts`
时间戳伪造引擎：
- 导出 `forgeEditTrail(text, sessionDurationMs?)` → `EditTrailEntry[]`
  - 自动反推 2h 非线性编辑时间表（随机停顿、多次修改段落）
- 导出 `buildExportMarkdown(cleanText, trail)` → 含 `<!-- Edit v1.x Time: xxx -->` 注释的 Markdown 字符串
- 导出 `copyRichTextWithTrail(cleanText, trail)` → 写入剪贴板（纯文本 or HTML）

#### [MODIFY] `src/lib/gemini.ts`
新增对抗生成集群函数：
- `runAdversarialCluster(text, apiKey, onProgress)` → `GenerationVariant[]`
  - 并行调用 5 套风格底层指令（语音速记风、技术初稿风、失眠碎碎念风、风格迁移风、微博口语风）
  - 返回 5 个变体，含 promptStyle 标签
